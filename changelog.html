Nouvelles Fonctionnalités Majeures

Système de Tour Guidé Interactif

Nouveau Tour Guidé Complet

Nouveau système de découverte guidée de l'application avec navigation automatique :

  • Tour général : 18 étapes couvrant toutes les fonctionnalités principales
  • Tours thématiques : Devis, Raccourcis clavier, Import de données
  • Navigation automatique : Change d'onglet automatiquement pour expliquer chaque section
  • Sauvegarde de progression : Reprend là où vous vous êtes arrêté
  • Message de bienvenue : Détection automatique des nouveaux utilisateurs
  • Raccourci F2 : Lancement rapide du tour guidé à tout moment

Nouveau Bulles d'Aide Contextuelles

Interface intuitive avec bulles animées qui suivent votre progression :

  • Animation smooth avec effet de pulse
  • Positionnement intelligent près des éléments UI
  • Overlay sombre pour mettre en valeur l'élément actuel
  • Compteur de progression (Étape X/Y)

Raccourcis Clavier Optimisés AZERTY

Nouveau Support Complet Clavier AZERTY Français

Plus de 30 raccourcis clavier adaptés pour clavier AZERTY :

  • Navigation rapide : Ctrl+&é"'( pour accéder aux onglets principaux
  • Alt+1-9 : Navigation directe vers n'importe quel onglet par position
  • Ctrl+Tab / Ctrl+Shift+Tab : Cycle entre les onglets
  • Actions courantes : Ctrl+S (Sauvegarder), Ctrl+N (Nouveau), Ctrl+D (Dupliquer)
  • Devis & Factures : Ctrl+Q (Nouveau devis), Ctrl+B (Convertir en facture)
  • Recherche : Ctrl+K (Quick Add), Ctrl+F (Recherche)

Nouveau Modal d'Aide Raccourcis Redesigné

Interface claire et organisée pour visualiser tous les raccourcis disponibles :

  • Organisation par catégories (Navigation, Actions, Devis, Recherche, Utilitaires)
  • Affichage visuel des touches avec style 3D
  • Adaptation automatique au thème sombre
  • Indicateurs 🔒 pour les raccourcis restreints en mode invité
  • Accessible via F1 ou Ctrl+/ à tout moment

Amélioré Sécurité Mode Invité

Protection renforcée des raccourcis clavier en mode invité :

  • Blocage automatique des raccourcis vers Catalogue, Configuration, Fournisseurs, Clients
  • Navigation intelligente qui saute les onglets restreints avec Ctrl+Tab
  • Messages d'alerte clairs lors de tentatives d'accès non autorisées
  • Liste des raccourcis accessibles affichée dans l'aide

Import/Export Excel Avancé

Nouveau Templates Excel Professionnels

Ajout de 3 templates Excel optimisés pour faciliter l'import de vos données :

  • Template Clients : Import en 1 clic de vos fichiers clients (nom, adresse, email, téléphone, historique)
  • Template Fournisseurs : Gestion complète de vos fournisseurs (coordonnées, délais, conditions de paiement)
  • Catalogue Matériaux : Import massif de matériaux avec métadonnées riches (prix, TVA, unités, conversions)

Gain de temps : Importez des centaines d'entrées en quelques secondes au lieu de les saisir manuellement.

Nouveau Export Excel Enrichi

Nouveau système d'export Excel avec formatage professionnel :

  • Export des devis et factures au format .xlsx avec mise en forme
  • Export des statistiques CA avec graphiques intégrés
  • Export des listes clients/fournisseurs/matériaux pour analyse
  • Support des formules Excel natives pour calculs dynamiques

Système de Relances Automatiques

Nouveau Relances de Paiement Intelligentes

Module complet de gestion des relances de paiement avec :

  • Détection automatique des factures impayées et échues
  • Système de relances programmables (J+7, J+14, J+30)
  • Templates d'emails personnalisables selon le niveau de relance
  • Historique complet des relances par client
  • Envoi automatique ou manuel selon vos préférences

Nouveau Configuration SMTP Améliorée

Interface simplifiée pour configurer vos serveurs d'envoi d'emails :

  • Support multi-providers (Gmail, Outlook, SMTP personnalisé)
  • Test de connexion en 1 clic
  • Gestion des pièces jointes (PDF devis/factures)
  • Prévisualisation avant envoi

Dashboard Analytics Amélioré

Amélioré Nouveaux Graphiques Temps Réel

Le tableau de bord a été entièrement repensé avec :

  • Graphique CA mensuel : Évolution du chiffre d'affaires avec comparaison N vs N-1
  • Top 20 produits : Classement par CA avec indicateurs de tendance
  • Répartition par catégorie : Camembert interactif des ventes par catégorie
  • Taux de conversion : Suivi du ratio devis→factures avec historique
  • Factures en retard : Alertes visuelles pour les impayés

Nouveau Recommandations IA

Système de recommandations intelligentes basé sur l'analyse de vos données :

  • Alertes stock bas pour les produits à forte rotation
  • Suggestions d'optimisation des marges
  • Détection automatique des tendances produits
  • Prévisions de ventes basées sur l'historique

Corrections de Bugs

Bugs Corrigés

Corrigé Calcul TVA

Problème : Certains matériaux affichaient "NaN%" au lieu du taux de TVA
Cause : Le système gérait uniquement le format tvaRate: 0.20 et non tva: 20
Solution : Support des deux formats avec détection automatique

Corrigé Export PDF Multi-pages

Correction du bug où les devis de plus de 20 lignes ne s'exportaient pas correctement en PDF multi-pages.

Corrigé Synchronisation Cloud

Résolution des conflits de synchronisation lors de modifications simultanées sur plusieurs postes.

Corrigé Scanner Code-Barres

Amélioration de la détection des codes EAN-13 avec checksum invalide.

Corrigé Thème Minimal Nordic

Correction des contrastes de couleurs pour meilleure lisibilité selon les standards WCAG 2.1.

Corrigé Calculs Arrondis

Les quantités calculées sont désormais systématiquement arrondies au supérieur pour éviter les manques.

Corrigé Recherche Caractères Spéciaux

La recherche gère maintenant correctement les caractères accentués et spéciaux (€, ², ³, °).

Corrigé Export FEC Comptable

Correction du formatage des dates pour conformité stricte avec le format réglementaire français.

Corrigé Bug Overlay Tour Guidé

Problème : Les overlays sombres s'empilaient lors de lancements multiples du tour guidé, rendant l'écran progressivement noir
Solution : Vérification et réutilisation de l'overlay existant au lieu d'en créer un nouveau à chaque lancement

Corrigé Backdrops Bootstrap Multiples

Correction du bug où les backdrops des modals Bootstrap s'accumulaient lors d'appuis multiples sur F1, rendant l'interface de plus en plus sombre.

Corrigé Navigation Tour Guidé

L'étape "Quick Add" du tour général naviguait vers l'onglet Accueil au lieu de Recherche. Corrigé pour une démonstration cohérente.

Corrigé Affichage Touches Clavier

Les touches de raccourcis (kbd) étaient quasi invisibles dans le tour guidé avec un fond blanc sur blanc. Ajout d'un style 3D avec ombre et contraste amélioré.

Migration & Installation

Migration depuis v3.1.0

Cette mise à jour est 100% compatible avec la version 3.1.0. Aucune action manuelle requise.

  1. Télécharger AutoCalc OptiDevis Setup 3.2.0.exe
  2. Fermer l'application si elle est ouverte
  3. Lancer l'installateur et suivre les instructions
  4. Au redémarrage, vos données (devis, factures, clients) sont automatiquement préservées

Recommandation : Effectuez un backup manuel via le menu Admin avant toute mise à jour majeure.
