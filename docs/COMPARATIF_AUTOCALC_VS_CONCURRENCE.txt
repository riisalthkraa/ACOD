        1. PRÉSENTATION D'AUTOCALC OPTIDEVIS v3.1.0
═══════════════════════════════════════════════════════════════════════════════

AutoCalc OptiDevis est une solution de gestion professionnelle complète
destinée aux entreprises de construction et au négoce de matériaux.

TYPE : Application desktop (Electron) multi-plateforme
CIBLE : PME, Artisans, Négociants en matériaux de construction
MODÈLE : Licence unique (pas d'abonnement)
DONNÉES : Locales (souveraineté totale)

ARCHITECTURE TECHNIQUE :
├─ Frontend : JavaScript ES6 modulaire, HTML5, CSS3, Bootstrap 5.3.3
├─ Backend : Electron v38.3.0
├─ Base de données : JSON (lowdb v3.0.0) avec système de cache
├─ Tests : 140 tests automatisés (Jest)
└─ Qualité : Audit 72/100, ESLint 0 erreur

MODULES PRINCIPAUX (20) :
✓ Authentification et permissions (Vendeur/Admin)
✓ Catalogue matériaux avec métadonnées riches
✓ Gestion clients et fournisseurs
✓ Création de devis avec calculs intelligents
✓ Facturation et suivi des paiements
✓ Tableaux de bord analytics avancés
✓ Export multi-formats (PDF, Excel, FEC comptable)
✓ Système de backup/restauration
✓ Scanner code-barres
✓ Saisie rapide en langage naturel
✓ 28 thèmes personnalisables
✓ 5 designs PDF professionnels

✓ CALCULS INTELLIGENTS AUTOMATIQUES
  ─────────────────────────────────────────────────────────────────────────
  Exemple : "75m² plaque BA13"
  → Détecte piecesPerM² = 1.05
  → Calcule automatiquement : 75 × 1.05 = 78.75
  → Arrondit au supérieur : 79 plaques
  → Gain de temps : 90% vs saisie manuelle

  Gère automatiquement :
  • Conversions m² → pièces (plaques, carreaux...)
  • Conversions m³ → kg → sacs (béton, ciment...)
  • Conversions m → unités (rails, profiles...)
  • Arrondis intelligents (toujours au supérieur)

✓ SAISIE RAPIDE EN LANGAGE NATUREL
  ─────────────────────────────────────────────────────────────────────────
  Tapez : "50 parpaings" → Ajout immédiat au devis
  Tapez : "10m³ béton" → Calcul auto sacs nécessaires
  Tapez : "20m rail placo" → Conversion auto en quantité

  • Recherche fuzzy 5 niveaux (tolère fautes frappe)
  • Reconnaissance unités (m², m³, m, unités, /h)
  • Sélection auto du matériau le plus pertinent
  • Vitesse création devis : 3x plus rapide

✓ ANALYTICS AVANCÉS AVEC IA
  ─────────────────────────────────────────────────────────────────────────
  • Détection automatique des tendances produits
  • Recommandations intelligentes (réappro stock, marges...)
  • Alertes prédictives (stock bas, factures échues...)
  • Top 20 produits par CA avec graphiques Chart.js
  • Comparaison mois actuel vs mois précédent
  • Taux de conversion devis→factures

✓ SCANNER CODE-BARRES INTÉGRÉ
  ─────────────────────────────────────────────────────────────────────────
  • Scan via webcam (codes 1D : 128, 39, EAN...)
  • Recherche auto du matériau dans le catalogue
  • Ajout immédiat au devis
  • Idéal pour gestion entrepôt/négoce
