Comment utiliser le template ?

  1. Téléchargez le fichier Excel ci-contre
  2. Remplissez les colonnes avec vos données clients
  3. Respectez le format des champs (dates, emails, codes postaux)
  4. Dans AutoCalc, allez dans Admin → Clients → Importer
  5. Sélectionnez votre fichier et validez
  6. Vos clients sont automatiquement créés !

Points d'attention

  • Les emails doivent être valides et uniques
  • Les champs marqués * sont obligatoires
  • Format date : JJ/MM/AAAA (ex: 24/10/2025)
  • Ne modifiez pas les en-têtes de colonnes

Calculs Intelligents

Grâce aux métadonnées riches, AutoCalc OptiDevis calcule automatiquement les quantités nécessaires :

Exemple : Plaques BA13
Vous saisissez : "75m² plaque BA13"
→ Détecte piecesPerM² = 1.05
→ Calcule : 75 × 1.05 = 78.75
→ Arrondit au supérieur : 79 plaques
✓ Gain de temps : 90%
Exemple : Béton
Vous saisissez : "10m³ béton"
→ Détecte density = 2400 kg/m³
→ Calcule : 10 × 2400 = 24000 kg
→ Convertit : 480 sacs de 50kg

Champs de métadonnées

coverage Surface/volume couverte par unité (m²/m³)
piecesPerM² Nombre de pièces au m² (ex: 1.05 pour BA13)
density Densité en kg/m³ (pour béton, ciment...)
unitWeight Poids unitaire (pour sacs, cartons...)
